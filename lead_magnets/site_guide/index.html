None of that. Below is the complete process — every prompt and command — to go from a blank folder to a live website on the internet in one sitting, with Claude Code doing all the engineering.

0Setup (10 minutes, one time)

  • Install Node.js from nodejs.org (18+), then Claude Code: npm install -g @anthropic-ai/claude-code
  • Create a free Vercel account at vercel.com (free tier is genuinely enough — SSL, global hosting, custom domains)

1Describe your business — Claude builds the site

Open a terminal in an empty folder, run claude, and paste:

Prompt Build a landing page for [YOUR BUSINESS — one plain-English sentence about what you sell and to whom]. I want: a hero section with a strong headline, 3 feature cards, a pricing section, and a footer. Warm modern design, mobile responsive. Use plain HTML and CSS in a single file, then start a local preview server and give me the URL.
Design taste, cheap: add "in the style of stripe.com" (or any site you love) to the prompt. Claude knows what those look like and borrows the vibe, not the code.

2Review in your browser — change it by talking

Open the preview URL Claude gives you. Then just say what you see wrong:

Prompts (real examples) Make the headline bigger and the orange darker. · Add a testimonials section with 3 quotes. · The pricing feels hidden — move it above the footer and make the number huge.
Rule: one change per prompt, look, then next. You're the art director; Claude is the (very fast) developer.

3One command puts it live

Prompt Deploy this site to Vercel production. Walk me through login if needed.
✓ Build completed ✓ SSL certificate issued 🌍 Live: https://yourbusiness.vercel.app · cost: ₹0

That URL is a real website — anyone in the world can open it, on any device, instantly.

4Optional finishing moves (each is one prompt)

  • Custom domain: "Connect mybusiness.com to this Vercel project — walk me through the DNS records."
  • Lead capture: "Add an email signup form that saves to a Google Sheet."
  • WhatsApp button: "Add a floating WhatsApp chat button linked to my number."
  • SEO basics: "Add proper page titles, meta descriptions and an OG image so links preview well."

The 5 mistakes that stall beginners

  • Asking for the whole site in one giant prompt — hero first, then one section at a time
  • Describing design in developer words — say "feels cramped" or "too corporate"; Claude translates
  • Skipping the phone check — open the live URL on your phone; most visitors are mobile
  • Fighting errors yourself — paste any error straight back into Claude Code; it fixes its own mess
  • Polishing before publishing — deploy the 80% version today; iterate on the live site tomorrow
