# Templates, Value Mechanics & Mistakes

## The 3-Act timing (reel)
- **Act 1 — Hook:** 0-3s. Stop the scroll.
- **Act 2 — Value:** 3-25s. Deliver on the hook's promise.
- **Act 3 — CTA:** last 3-5s. One clear next step.

## Act 2 value mechanics (the core craft)

**Inverted Pyramid** — order your points: 2nd-best first, best second, rest
descending. Starting good-then-better creates an upward feeling that keeps people
watching. Starting with your best makes everything after feel like a letdown.

**Value Loop** — every point gets three beats:
- *What it is* — explain simply (<5s)
- *How to do it* — the actual tactical step
- *Why it matters* — tie it to their goal / what they lose without it

**Micro-hooks** — a teaser line every 8-10s, placed before each new point, to
re-grab attention before the natural drop-off:
- "But here's what most people miss…"
- "Stay with me, the last one matters most…"
- "Wait, it gets better…"
- "This next part is the important one…"

**Condensation rules:**
- One-Sentence Rule: cut any sentence the video survives without.
- 10-Second Rule: each point digestible in ≤10s.
- Clarity Test: a 12-year-old could follow it.

## Advanced retention devices (layer one, optional)
- **In Medias Res:** open mid-story at the dramatic moment, then circle back.
- **Breadcrumb:** tease point 3 while delivering point 1.
- **Retention Hook Ladder:** "3 things, but #3 will change everything" → deliver in
  ascending hype.
- **Callback:** plant something early, pay it off at the end to force a rewatch.

## CTA logic
- **60-Video Rule:** for a young account, first ~60 posts use soft CTAs (follow /
  save / share) to build trust; introduce hard conversion CTAs after.
- **Scale by content type:** Virality → follow/save. Authority → save. Conversion →
  comment-keyword (highest converting; feeds the ManyChat funnel).
- **Native integration:** solve a pain point, then offer the resource as help, not a
  pitch. e.g. "Most founders get overwhelmed setting this up — comment SYSTEM and I'll
  send you the exact steps."

## The 7 script templates

1. **Problem-Solution** — Hook (stop doing X) → expand the problem → 1-3 fix steps →
   proof → CTA.
2. **Myth-Busting** — Hook (everyone says X, they're wrong) → why that advice fails →
   the truth → proof → CTA.
3. **Behind-the-Scenes** — Hook (what really happened when I…) → setup → conflict →
   resolution → lesson.
4. **List / Tips** — Hook (N things I wish I knew before…) → tip 1 → tip 2 → tip 3 →
   summary → CTA.
5. **Transformation** — Hook (X ago I was [bad], now I'm [good]) → before → turning
   point → process → current state.
6. **Question-Answer** — Hook (the #1 question I get…) → why people ask → the answer →
   the insight most miss → CTA.
7. **Mistake-Lesson** — Hook (this mistake cost me X) → what I did wrong →
   consequences → lesson → do-this-instead.

## Common mistakes (self-check, reject if present)
- Weak hook ("Hey guys", "In this video", "What's up").
- Information overload (more than 3 points).
- Generic value ("just be consistent") with no unique angle.
- Weak or missing CTA ("thanks for watching").
- Best point placed first (breaks the inverted pyramid).
- No micro-hooks → mid-video drop-off.
- Topic drifting outside the brand's niche/pillars.
- Script that only works with sound (must work on mute).
