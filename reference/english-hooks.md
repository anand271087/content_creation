## Hook Selection Guide

Use this table to pick the best hook category based on your content goal.

| Content Goal | Best Hook Numbers | Why |
|---|---|---|
| Introduce a new AI tool | #176–#190, #51–#75 | Tool reveals + curiosity about unknown capability |
| Warn about a mistake or bad habit | #161–#175, #1–#25 | Loss aversion drives immediate attention |

| Build authority / status positioning | #26–#50, #101–#125 | Identity + credibility through specificity |
| Compare old vs new workflow | #126–#140, #76–#100 | Contrast triggers cognitive dissonance |
| Share a personal story or confession | #141–#160 | Vulnerability + relatability drives trust |
| Drive saves / bookmarks | #101–#125, #176–#190 | Specific numbers + actionable methods = save-worthy |
| Drive comments / debate | #126–#140, #191–#200 | Comparison + challenge creates opinion-sharing |
| Grow audience (broad TAM) | #1–#25, #51–#75 | Fear/loss + open loops stop the widest audience |
| Sell a course / offer | #76–#100, #26–#50 | Transformation + identity make the offer obvious |
| Build urgency / FOMO | #1–#25, #161–#175 | Loss + warning creates immediate pressure |
| Share a process / tutorial | #176–#190, #109–#125 | Tool + specific numbers = high trust, high save rate |
| Target non-tech audience | #141–#160, #76–#100 | Story + transformation = accessible entry point |
| Target high-earning professionals | #26–#50, #126–#140 | Status + comparison speaks to identity |
| Challenge skeptics | #191–#200, #161–#175 | Direct dare + warning breaks through resistance |
